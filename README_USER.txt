============================================
   SQL Workbook Generator
   One-Click Setup Guide
============================================

🔹 FOR NORMAL USERS (NO TERMINAL NEEDED):

1. DOUBLE-CLICK the file:
   - Windows: launch_portal.bat
   - Mac: launch_portal.command
   - Linux: launch_portal.sh or start.py

2. A terminal window will open.
3. Wait for "Portal is running!" message.
4. Your browser will open automatically.

🔹 WHAT YOU CAN DO:

📄 Generate Workbooks
   - Choose SQL file
   - Select number of questions
   - Click "Generate Workbook"

🎯 Interactive Practice
   - Choose difficulty levels
   - Practice one question at a time
   - Get instant feedback with scoring

🗄️ Generate Database
   - Create practice database
   - Realistic sample data
   - 5 tables with 100 rows each

📤 Upload Files
   - Upload custom SQL files
   - Upload Python models

🔹 TROUBLESHOOTING:

❌ "Port already in use"
   → The script will automatically try another port

❌ "PostgreSQL not running"
   → The script will start it automatically

❌ "Dependencies not found"
   → The script will install them automatically

❌ "Permission denied"
   → Right-click and select "Run as executable"
   → Or run: chmod +x launch_portal.sh

🔹 SYSTEM REQUIREMENTS:
   - Python 3.8+
   - PostgreSQL (will be installed if missing)
   - Internet connection (for first-time setup)
